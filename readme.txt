== Description ==

This Plugin lets you redirect the visitors to a specific URL when the page/post is viewed from a mobile device. You can specify on which URL you want the visitors to be redirected for particular page. You can set different URLs for different pages/posts. You can also specify in the Page edit options if you want the redirection for particular page or not.

The plugin can be useful when your page is only for desktop device and you can show warning or information page to the mobile visitors telling them that it's a desktop only version. It could be also useful when you don't want some page to be viewed on mobile devices. There can be many other uses as well, It depends on you want to use it.

== Installation ==

1. Download the plugin from Wordpress Plugins screen or Upload the plugin using the "Add New Plugin" Page.
2. Activate the Plugin.
3. Go to edit any of your existing page or create new, You'll see a custom meta box named "Mobile Redirect".
4. Click the Checkbox and enter the redirection url if you want that page to be redirected when visited from mobile.
5. That's it, Enjoy!
